# CSV エクスポート

## 目的・背景

管理画面の一覧データを CSV でダウンロードできるようにする。Excel で開くユーザーが多い。

## 要件

- 一覧データ（文字列フィールドの 2 次元配列）を CSV 文字列に変換できる
- Excel でそのまま開ける形式にする

## 仕様

- RFC 4180 に準拠する
  - フィールドにカンマ・改行・ダブルクォートを含む場合、フィールド全体をダブルクォートで囲む
  - フィールド内のダブルクォートは `""` に重ねてエスケープする
  - 行区切りは CRLF（`\r\n`）
- 先頭にヘッダー行を出力する

## 決定事項

- 2026-06-30: RFC 4180 準拠とする（Excel 互換のため）
- 2026-06-30: 文字コードは UTF-8（BOM の扱いは呼び出し側の責務とし、本モジュールでは付与しない）
